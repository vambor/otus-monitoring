# Домашнее задание - Формирование dashboard на основе собранных данных с Grafana
## Описание
Установлена grafana версии 12.2.1 docker
- В качестве источника данных выбран настроенный локально Prometheus 
- Созданы папки infra для дашбордов инфраструктуы и app для дашбордов приложения
- В папке infra создан дашборд Infra Overview отображающий: загрузку ЦПУ, процент использования оперативной памяти, заполненность диска, сетевой трафик.
- В папке app создан дашборд App Overview отображающий: доступность CMS (на основе метрики probe_success от BlackBox экспортера) и время отклика.
